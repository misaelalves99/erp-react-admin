# ERP Admin Pro — React 19 + TypeScript + React-admin (Open Source)

ERP administrativo (Financeiro, Compras, Vendas, Estoque, Fiscal) com UI dark premium e arquitetura moderna.

## Rodar
```bash
npm i
npm run dev
```

## Login (demo)
- Email: `admin@erp.local`
- Senha: `admin`

## Registro (demo)
Acesse `/register` para criar um usuário local (salvo em `localStorage`).

# Arquitetura (visão geral)

- `src/design/tokens`: identidade visual (cores, tipografia, espaçamentos, radius, shadows, breakpoints)
- `src/ui`: componentes reutilizáveis (PageHeader, MetricCard, StatusChip, etc.)
- `src/core`: hooks, contexts e utils
- `src/data`: dados mock (in-memory) e montagem do "db" para o DataProvider
- `src/admin`: app React-admin (layout, providers, resources, fields, dashboard, pages)
